Code of Conduct

Importance

This Code of Conduct explains the conduct required from AWS community members and event participants. This Code of Conduct is non-negotiable. Any behavior that runs counter to this Code of Conduct will not be tolerated.

Scope

All community members and event participants (including AWS employees, attendees, vendors, sponsors, speakers, volunteers, and guests) must comply with this Code of Conduct. This Code of Conduct applies to the main event and all related events, social or otherwise.

Behavior

To ensure that AWS remains a community where members and event participants feel welcome and safe, all community members and event participants must:

  • Help create a safe and supportive environment for all event participants.
  • Comply with the instructions from the event and venue staff.
  • Not engage in any disruptive speech or behavior.
  • Not interfere with the event or another individual's participation in the event.
  • Not engage in any form of harassment, offensive, discriminatory, or threatening speech or behavior, including but not limited to, relating to race, gender, gender identity and expression, national origin, religion, disability, marital status, age, sexual orientation, military or veteran status, or other protected category.
  • Comply with all applicable laws.

Consequences

Any member or event participant not complying with this Code of Conduct may be denied entry or required to leave from any and all events, which AWS Community Day PH 2023 Volunteer organizers may determine at their sole discretion.

There will be no refund of any fees in case of any denial of entry or demand to leave any event.

Any violation of this Conduct of Conduct by any member or event participant may result in its/his/her/their disqualification from future AWS events.

AWS Community Day PH 2023 Volunteer organizers may contact local law enforcement if they deem appropriate.

Contact

If you have any concern regarding this Code of Conduct, please promptly contact AWS User Group Philippines organizers at info@awsug.ph
